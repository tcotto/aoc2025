# Java Template

This is a minimal Java template for creating small Java applications.

## Files

- `Main.java` - The main entry point for the application

## How to Use

### Compile the program:
```bash
javac Main.java
```

### Run the program:
```bash
java Main
```

## Customization

1. Modify the `main` method in `Main.java` to implement your solution
2. Add additional Java files as needed in the same directory
3. Compile and run using the commands above
